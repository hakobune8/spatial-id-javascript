# 空間ID試行環境

## 概要
本ディレクトリは、空間ID共通ライブラリをブラウザ上で簡易に試すための試行環境です。

ズームレベル、標高、経度、緯度を入力することで、対応する空間ID（z/f/x/y）を算出し、
その対象範囲を地図上に表示します。

---

## 提供機能
- 座標（緯度・経度・標高）およびズームレベルから空間IDを算出
- 空間IDに対応する範囲を地図上に可視化

---

## 公開ページ
https://aoshima-yuki.github.io/ouranos-gex-lib-for-JavaScript/

---

## 実装概要
本画面は、空間ID共通ライブラリ（src/配下）をビルドして生成された
配布用ライブラリ（UMD形式のJavaScript）を利用し、
GUI上で空間IDの算出および範囲取得を確認できるようにしたものです。  

環境構築を行うことなく、ブラウザ上で空間ID共通ライブラリの一部機能を利用できる構成としています。

---

## ライブラリ構成の整理

本試行環境で利用している空間ID共通ライブラリは、以下の2層で構成されています。

### 本体ライブラリ（src/配下）
- TypeScriptで実装された開発用コード  
- 空間IDの計算ロジック（座標変換、タイル計算等）を含む

### 配布用ライブラリ（UMD形式のJavaScript）
- 本体ライブラリをビルドして生成された実行用コード  
- `vendor/spatialid/index.js` として配置  
- ブラウザから直接読み込み可能

本試行環境はGitHub Pages上で動作するため、
ブラウザで直接実行可能なJavaScript形式（UMD）を利用しています。
